# Resume Tailoring Rules — Industry-Specific Rewriting Guide

## Universal Rules

### The STAR Reframe
Every experience bullet should answer: Situation → Task → Action → Result

- **Before**: "负责用户增长的数据分析"
- **After**: "通过用户行为数据分析，识别出3个关键流失节点，推动产品优化后月留存提升12%"

### Quantify Everything Possible
- "写了很多报告" → "撰写15+份周度数据分析报告"
- "提高了效率" → "将数据处理流程从4小时缩短至30分钟"
- "管理社群" → "运营500+人技术社群，周活跃率60%"

### Action Verb Upgrade
| Weak | Strong |
|------|--------|
| 负责 (responsible for) | 主导 (led), 推动 (drove) |
| 参与 (participated) | 贡献 (contributed), 交付 (delivered) |
| 帮忙 (helped) | 支持 (supported), 协作 (collaborated) |
| 做了 (did) | 设计并实现 (designed and implemented) |
| 了解 (know) | 掌握 (proficient), 实践 (practiced) |

## Industry-Specific Rules

### Tech / Software Engineering
- Focus on: tech stack, system design, scale (users/QPS), impact metrics
- JD keywords: language names, frameworks, cloud platforms, architecture patterns
- Bullet format: `[Built/Designed/Optimized] [system/feature] using [tech stack] that [impact metric]`
- Example: "Designed and implemented a real-time notification service using Go and Redis, handling 50K+ events/day"

### Data / Analytics
- Focus on: data volume, tools, insights generated, business impact
- JD keywords: SQL, Python, Tableau, A/B testing, statistical methods
- Bullet format: `[Analyzed/Modeled] [data type] using [tool] to [business outcome]`
- Example: "Analyzed 2M+ user sessions using Python and SQL to identify churn predictors, informing a retention strategy that reduced churn by 8%"

### Product / PM
- Focus on: user research, roadmap, cross-functional coordination, metrics driven
- JD keywords: PRD, user stories, A/B testing, stakeholder management, OKRs
- Bullet format: `[Led/Defined/Drove] [initiative] that [user/business outcome]`
- Example: "Defined product requirements for a payment optimization feature, coordinating across engineering, design, and compliance teams to ship on time"

### Finance / Consulting
- Focus on: deal size, clients, models built, accuracy, process improvement
- JD keywords: financial modeling, due diligence, valuation, forecasting, Excel/VBA
- Bullet format: `[Built/Analyzed/Advised] [scope] resulting in [quantified outcome]`
- Example: "Built a 3-statement financial model for a Series B SaaS client, supporting a $20M fundraise"

### Marketing / Operations
- Focus on: channels, budget, ROI, growth metrics, campaigns
- JD keywords: SEO/SEM, CRM, content strategy, funnel optimization, ROAS
- Bullet format: `[Ran/Optimized/Grew] [channel/campaign] achieving [metric]`
- Example: "Optimized Google Ads campaigns across 50+ keywords, reducing CAC by 25% while maintaining conversion volume"

## Chinese Resume Specifics

### 中文简历语言升级
| 普通表述 | 优化表述 |
|---------|---------|
| 用Python做数据分析 | 基于Python(Pandas/NumPy)完成XX场景的数据分析 |
| 和同学一起做项目 | 与4名团队成员协作完成XX项目 |
| 在XX公司实习 | 于XX公司担任XX职位，负责XX模块 |

### Common JD Phrases → Real Meaning
| JD原文 | 实际含义 | 简历对应策略 |
|--------|---------|------------|
| 抗压能力强 | 加班多/节奏快 | 如有赶项目经历，强调"在紧迫时间线内交付" |
| 具备owner意识 | 需要自驱力 | 强调"独立负责"、"从0到1"的经历 |
| 良好的沟通能力 | 跨部门协作 | 强调"与XX部门协作"的经历 |
| 数据驱动 | 需要数据思维 | 所有经历都尽量加入量化结果 |
| 结果导向 | 看重产出 | 用结果开头："提升了XX"、"降低了XX" |
