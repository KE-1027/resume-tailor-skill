# Industry Term Mappings — JD Language Decoder

## What JDs Really Mean

### "The Hype Detector"

| JD Phrase | Literal Meaning | Resume Strategy |
|-----------|----------------|-----------------|
| Fast-paced environment | High pressure, tight deadlines | Show experience with tight timelines |
| Wears many hats | Small team, broad responsibilities | Highlight diverse skill set |
| Self-starter | Minimal onboarding/supervision | Show "独立完成" or "从0到1" projects |
| Detail-oriented | They've been burned by sloppy work | Show examples of careful, precise work |
| Rock star / Ninja / Guru | Overwork expected, under-market pay | Be cautious — verify company culture |
| 10x engineer | Unrealistic expectations | Show strong metrics but be realistic |
| Growth mindset | Willing to learn on the job | Show self-learning examples |
| Hit the ground running | No training provided | Emphasize relevant experience |
| Dynamic | Chaotic | Show adaptability through varied experiences |
| Lean team | Understaffed | Frame breadth of experience positively |

### "The Responsibility Translator"

| JD Says | Actually Means |
|---------|---------------|
| Manage end-to-end | You own everything, including blame |
| Collaborate with stakeholders | Lots of meetings |
| Drive cross-functional initiatives | Herding cats across departments |
| Champion best practices | Fix the mess they've accumulated |
| Build from scratch | No existing systems or documentation |
| Maintain and optimize legacy systems | Deal with decade-old spaghetti code |
| Customer-obsessed | Work weekends when customers complain |
| Move fast and break things | We ship buggy code, you fix it later |

### Skill Bundles by Role

**Frontend Developer**:
Core: HTML5, CSS3, JavaScript (ES6+), Git, Responsive Design
Framework: React / Vue / Angular (one deep, one aware)
Plus: TypeScript, Webpack/Vite, Testing (Jest/Cypress), Browser DevTools

**Backend Developer**:
Core: One backend language (Python/Java/Go/Node), SQL, REST APIs, Git
Framework: Django/Spring/Express/FastAPI
Plus: Docker, Redis, Message Queues, System Design, Testing

**Data Analyst**:
Core: SQL (window functions, CTEs), Excel/Google Sheets, Data Visualization
Tools: Tableau/Power BI/Looker, Python (Pandas) or R
Plus: A/B Testing, Statistical Methods, Business Acumen

**Machine Learning Engineer**:
Core: Python, SQL, ML Fundamentals, One framework (PyTorch/TensorFlow)
Plus: MLOps, Feature Engineering, Model Deployment, A/B Testing

**Product Manager**:
Core: User Research, PRD Writing, Data Analysis, Roadmap Planning
Plus: SQL (basic), A/B Testing, UX Principles, Technical Communication

**DevOps / SRE**:
Core: Linux, One Cloud (AWS/Azure/GCP), CI/CD, Docker, Scripting
Plus: K8s, Terraform, Monitoring (Prometheus/Grafana), Incident Response
